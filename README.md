# Uber Analytics Dashboard 🚖📊

## Overview
This project presents an interactive analytics dashboard for Uber operations.  
It highlights key performance metrics such as bookings, revenue, ratings, and travel distance, along with detailed insights into vehicle types, payment methods, and pickup locations.

## Features
- **Revenue Over Time**: Monthly revenue trends (Jan–Dec).  
- **Booking Status**: Completed vs Incomplete rides.  
- **Pickup Locations**: Top demand hotspots.  
- **Vehicle Types**: Distribution across Go Mini, Sedan, XL, eBike, etc.  
- **Payment Methods**: Cash, Card, Wallet, UPI usage.  
- **KPIs**: Total bookings, revenue, ratings, and distance traveled.

## Tech Stack
- Power BI for visualization  
- Data preprocessing with Python (Pandas/NumPy)  
- Dataset: Simulated Uber booking & revenue data  

## How to Use
1. Clone this repository  
2. Open the `.pbix` file in Power BI  
3. Explore interactive charts and filters
